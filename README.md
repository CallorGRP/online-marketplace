# 🛍 온라인 마켓플레이스
</br>

## 🛒 개요
- 쿠팡을 클론하여 e-commerce 서비스의 서버를 설계하고 구축합니다.
- 이 프로젝트는 다음의 **목표**를 갖습니다.
  - 대규모 트래픽과 대용량 데이터 처리가 가능하도록 합니다.
  - 유지보수성을 고려하여 객체지향설계에 따라 개발하고, 테스트코드를 작성합니다.
  - CI/CD 를 구성하여 빌드/테스트/배포를 자동화합니다.
